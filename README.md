# 📘 Objetos en JavaScript

Este proyecto muestra cómo trabajar con **objetos en JavaScript**, utilizando **clases**, **funciones constructoras**, y métodos como `call` para cambiar el contexto de ejecución de una función.

---

## 🧠 Conceptos cubiertos

- **Objetos:** Agrupación de datos y funcionalidades en una sola entidad.
- **Clases y Constructores:** Uso de `class` y funciones constructoras para crear múltiples instancias de objetos.
- **Herencia de métodos:** Implementación de métodos en clases.
- **Método `call`:** Ejecución de un método de un objeto con el contexto de otro objeto.
